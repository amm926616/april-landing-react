## ✨ Features

- 🎧 Smooth and reliable local music playback
- 🪄 Clean and minimal interface with light/dark theming
- 📁 Simple library management and folder selection
- 📃 Real-time synced `.lrc` lyrics display
- 🚀 Fast startup with encrypted configuration using `EasyJson`
- 🛡️ Lightweight, private, and secure — no background tracking

---

## 🌱 Why April?

April Music Player is handcrafted for simplicity and elegance.
It’s a personal project meant for those who:

- want something *fast and clutter-free*
- love music without distractions
- appreciate thoughtful design
- believe in tools that “just work”
